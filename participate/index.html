About the research

Hello, I'm Andrew, a PhD student and lecturer at University of Chester, and have developed a tool named JOOFT to assist with the provision of automated feedback to students learning Java, in particular, aspects related to designing objects.

JOOFT has recently been integrated into a web-based tutorial, where students can write Java code following a tutorial and test it as they go. The project allows students to run tests to check their code at various stages and gives feedback on erroneous code. The task is designed to introduce concepts of object-orientation through the design a simple class.

The tool is open source and, having trialled the tool in the context of a lab task with students at my own institution, I'm looking invite other tutors who teach Java to review the task and tool.

Tutors are welcome to use the tool in their own teaching also, if they wish.
